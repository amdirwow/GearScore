# Q: Як GearScore позначає відсутню модель персонажа?

## Answer

У центрі області PlayerModel додано GS_ModelUnavailableText. Серверний META містить online=0/1: для офлайн-запису показується золотистий текст Персонаж офлайн, для онлайн-персонажа без доступного UnitToken — Модель недоступна. Коли SetUnit доступний, напис ховається і показується 3D-модель.
