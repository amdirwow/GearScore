# Q: Коли GearScore може показати модель онлайн-персонажа без SQL?

## Answer

Коли клієнт має UnitToken персонажа: player, target, mouseover, focus, party1-4, raid1-40 або arena1-5. GearScore_GetUnitTokenByName тепер перевіряє всі ці токени без урахування регістру та суфікса реалму; PLAYER_TARGET_CHANGED перемальовує вже відкритий профіль. Сам факт онлайн-статусу без доступного UnitToken не дозволяє PlayerModel:SetUnit побудувати модель за ім'ям.
